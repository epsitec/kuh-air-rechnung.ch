La facture QR du point de vue du développeur

La facture QR comporte un bulletin de versement qui est muni d'un code QR.

Les spécifications sont détaillées dans plusieurs documents publiés par SIX :

  • Implementation Guidelines suisses pour QR-facture.
  • Implementation Guidelines suisses pour les messages client-banque pour les virements dans le trafic des paiements (pain.001).

La syntaxe des données d'émetteurs (##S1) est définie par Swico. Ces données sont utiles aux logiciels des débiteurs : elles complètent le code QR avec des informations pertinentes sur la facture et la TVA.
